# StokerX
Control and Monitoring application for the Stoker BBQ Controller

Welcome to StokerX

You've got a Stoker. You've got a Mac. Now you can use them together!

StokerX is a monitoring and control application for the Stoker BBQ controller. Using StokerX, you can record the 
operation of your smoker, control the temperature setting for the blower, plot the temperature readings for both 
the pit and food probes, and be notified when temperature targets are achieved.

Features

Periodic or temperature alarm notifications via Email, Twitter, Growl, or Prowl (push notifications).
Time vs Temperature graph with annotations.
Customizable reports.
Built-in update mechanism.
Built-in crash and feedback reporter.
